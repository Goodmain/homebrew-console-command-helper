# homebrew-console-command-helper

Homebrew tap for [console-command-helper](https://github.com/Goodmain/console-command-helper).

## Install

```bash
brew tap Goodmain/console-command-helper
brew install cch
```

## Upgrade

```bash
brew upgrade cch
```

## Uninstall

```bash
brew uninstall cch
brew untap Goodmain/console-command-helper
```
